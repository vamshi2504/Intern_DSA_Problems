Add Two Numbers

You are given two non-empty linked lists that represent two non-negative integers. The digits are stored in reverse order, and each node contains a single digit.

Add the two numbers together and return the sum as a linked list in the same reverse-order format.

You may assume the two numbers do not contain leading zeros, except for the number 0 itself.

Examples:

Input: l1 = [2,4,3], l2 = [5,6,4]
Output: [7,0,8]

Input: l1 = [0], l2 = [0]
Output: [0]

Input: l1 = [9,9,9,9,9,9,9], l2 = [9,9,9,9]
Output: [8,9,9,9,0,0,0,1]

This problem is commonly solved by traversing both lists together while keeping track of carry.
